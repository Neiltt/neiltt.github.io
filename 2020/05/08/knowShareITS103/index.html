Spring boot set up

download tools & apache maven

https://spring.io/projects/spring-boot
https://maven.apache.org

generate

https://start.spring.io/
choose: Maven Project, Java, 2.1.14(SNAPSHOT)
Metadata: com.taxxx.queryInfo, QueryInfo, QueryInfo, Bag Info query, Jar, 8
Dependencies: Spring Web

setting

1.open STS Window->Preferences->Maven->User Settings:D:\apache-maven-3.5.0\conf\settings.xml
2.Run->Run Configuration->Environment->SERVER_PORT:8080…
3.choose Package(Project&right click)->Maven->Update Project
4.add package:com\taxxx\queryInfo\web\controller->java:HelloController
@Controller
@RequestMapping(“/queryInfo”)
public class HelloController
{
@RequestMapping(“/hello”)
@ResponseBody
public String hello() {
return “Hello World!”;
}
}

edit

edit resources\application.properties->server.servlet.context-path=/queryInfo(keyin)

Hello World

open http://localhost:8080/queryInfo/queryInfo/hello show “Hello World!”
