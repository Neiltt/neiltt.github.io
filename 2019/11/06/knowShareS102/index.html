時間管理

談到時間管理，部分人覺得這是成功人士才具備技能，說白的就是擁有良好習慣的生活作息，如此而已。

Start

時間是如何組成

想必不少聽到人談起，我什麼都沒做今天就結束了，這句話應該翻譯成『我一整天都耍廢爽爽過完了』，不過大多數的人會在最後一天，給自己一句我怎麼浪費一天，小小的內心譴責後，好像有反省就得過且過，事實上毫無反省改變。
既然時間如此容易流失，就應該要找出從哪裡流失的

先來看看這道數學式，非常簡單一天中，扣除睡覺、工作、吃飯、盥洗與處理家務，最後剩下3小時，就是你一天的自由時間，如果你還有上班通勤，下班打個遊戲、刷的視頻，基本上一天就這麼過了。

規劃大師與執行魯蛇

既然理解一天果然只有24小時，明顯我很忙的，除了上述歷程安排，我還有須多事情等著我去做，那就只能從其他地方借，壓縮原有的時間，譬如明天要交報告、要交提案，我來不及，沒關係我少睡一點，無形當中你做完了，交了報告，但作品質量如何，結果如何可想而知阿，借來的總該還的，這就是挖東牆補西牆的類型。
第二類人，他會計劃，他樂於計畫，往往與現實差異太大，而導致根本無法執行、或是計畫夭折，最慘莫過於發出豪語後，一個禮拜計畫依舊完美，但就是沒做。

上述幾種類型都擁有一項共通點，他們已計畫為主軸，計畫只是方法的一種，今天有了計畫，你能更理解順暢的完成工作，沒有你依舊可以完成，大家有沒有發現，執行完成才是主軸，搞棟這個思維，就能清楚的知道，任何計畫都是輔助你的執行的方法，當你要寫計畫，先腦海中模擬，你會如何執行，在以此設計相對應計畫。

挫折感

以往在計畫，也是搞不清楚，都已規畫為主角，容易把自己寫死，時間規劃堪稱完美，一分不差的無縫接軌，最後無疾而終，就是最簡單的問題，因為你根本還沒開始，當你克服這一關，才會開始嘗到，不了解自己的驚訝、過於樂觀的進程、拖延時程的低落感，以及自我放棄的良心不安，最終給自己不擅長規劃的標籤，我的人生沒有規劃的標籤。

堅持tips

執行主角，規劃輔助你，慢慢練習，循循漸進，自己當初也是完美計畫開始，耽誤1小時我的計畫就全毀，慢慢修正即使耽誤，依然將後續計畫完成，度過嘗試期後，開始會思索權衡，我剛始將主要的時段來做主要的事情，開始排列重要度的先後順序(這又是另外一門學問)，後期反而會因為事情事情規劃都上軌，而忘了勞役結合，讓自己太累，降低效率，而開始注重放鬆的安排，基本上其他時間就該幹嘛幹嘛。

工具安麗

chronodex
能將一整天時間，由一個圓餅圖清楚乾淨的呈現，不知道自己時間去哪朋友，乖乖畫2週就可以了。

番茄時鐘
專注訓練30分鐘，休息15分鐘，照這個節奏挺舒服的，疲勞感較不易累績。
