Introduction

Apache ActiveMQ為開放原始碼，面向消息的中介(MOM)，支援Java訊息服務 (JMS)1.1 版本，高效、可擴展的、較穩定的和安全消息通信。

ActiveMQ涉及方面

傳輸協議-無疑需要協議進行溝通，而提供廣泛的連接模式，其默認協議是openWire，埠號：61616

消息域- Point-to-Point (點對點),Publish/Subscribe Model (發布/訂閱者) (JMS結構模型)

消息存儲-部分重要訊息，儲到數據庫，避免異常遺失

Cluster (集群)-Network of brokers和Master Slave

Monitor (監控)-ActiveMQ一般由jmx來進行監控（Java Management Extensions，Java管理擴充)

補充

MOM（Message Oriented Middleware）是面向消息的中介軟體，提供 API 和管理工具。在發送消息之後，用戶端會繼續執行其他工作，並且在接收方收到消息之前，一直保留該消息。

Java訊息服務（Java Message Service，JMS）兩個應用程式之間，或分散式系統中傳送訊息，進行非同步通信。
