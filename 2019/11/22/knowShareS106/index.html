假裝無辜心理

其實想說是不認輸，但沒能力又不承認類人。

Start

Ｍr.S

社交當中，裝無辜有幾點作用，深藏與作球，除此以外的都是令人厭惡的行為。深藏當你遇到危險，把自己當白痴，欺騙獵人放過你；做球藉由做低身分，呈現無能為力，讓人幫助自己，做高別人身分幫定聯盟。其他的裝無辜只會顯得自己無能。

Ｍr.P

渴望得到同情，特別是自我同情，變相告訴自己很可憐，等待別人的施捨，心裡是比較好過，行為上就不會再努力了，得過且過直到無法負荷。

Ｍr.E

變相壓榨勞力，可以換句話說，能者多勞心理，無辜宣告你需要更多資源來辦成事，打上自身我不懂、團隊進程你需要教會我，於公於私來吸引更多的資源，惡性利益索取，短暫得利，長期則會崩盤。

淺見

無辜能用於短期逃避、檢視自我缺陷的好工作。
