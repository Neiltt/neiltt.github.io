恐懼從何而來？

恐慌與畏懼，自身安全遭遇到危險，或是遇到厭惡的事情，心跳加速、血壓上升之類的。

Start

小艾伯特實驗

1920年有個實驗「小艾伯特實驗」（Little Albert experiment）實驗中從嬰兒中挑選出9個月大小艾伯特，讓他接觸老鼠、兔子、狗、猴子、有頭髮和無頭髮的面具、棉絮。
兩個月後讓小艾伯特接觸白鼠，他不害怕，後續在小艾伯特接觸白鼠同時用鐵鎚敲擊鐵棒發生巨響，小艾伯特因巨響而哭，後面重複執行，最後小艾伯特見到白鼠，即使沒有巨響，其反應也是如聽到巨響一般，他學習了恐懼。實驗17天後，將一隻非白色兔子給他，同樣也讓他感受到不安。

Ｍr.S

群居生活，必定有階級制度，恐懼就是統治者手上的一把屠刀(權力)，同時其脖子上也是有一柄屠刀(喪失權力)，展現權謀某種方式罷了。

Ｍr.P

小艾伯特實驗，恐懼可以後天培養，相對應所有感情，反過來可以說情感是從人生中，觀察世界運行所成長出來的，源自於個人世界觀，如果你認為恐懼是不好的，那可以修補個人世界觀來補足，不過我到覺得恐懼是水，他會循環，不斷學習，從未知道已知，中間過程就是恐懼，只有一知半解才能感到害怕，學習想處更勝捨去與斷絕。
