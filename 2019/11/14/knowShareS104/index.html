購物欲求，我一定要買

購物是人活著最好證明，古代以物換物就是現在購物，差在於現代統一交易媒介，稱為貨幣，購物必須為了生存。

Start

Ｍr.S

能購代表你有某程度上的財富或地位，你具有影響力，大到公司老闆，小到你個人，起碼你在買時，賣方會對你笑。想彰顯牛，自然會買入高價品，搭建你所求的生活圈，如住在蛋黃區，自然不會與窮人有交集。，牛人購買遠遠不是常人能想想的。

Ｍr.P

你沒有你買，你沒有你也買，差異在於前者物質必需品，後者為心靈必需品。常人妄想可以透過物品，完美的補償缺陷，如自信感、自我滿足感、改變世界錯覺感，每天熬夜想讓遮瑕膏來掩蓋黑眼圈、顯逼買跑車，背上非你能環的車貸。本質上是你像獲得認可感，如果買入認可感零商品，就不要自圓其說。

Ｍr.E

相同品質下，當然價格漂亮更好，有幣就有價，價從市場來，股市中一檔股票，多人追價高，反之價低，形式上的價格為當時價格，考慮未來價格為價值，這是將時間軸加入，價值大於價格就可以買，反之不，買重在其價值，或是說這值不值錢。

淺見

我自己買不買倒是相當簡單，我看受者，買給誰用，然後會不會增加人的價值，增益還是增損，有機蔬果有益健康、演講有益認知增長、旅遊擴張感悟與價值，反之則不，功能性與重複性高的就不需要多，對自己誠實是蠻重要的練習。
