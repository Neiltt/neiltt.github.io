心靈認知

作者 Neil 日期 2019-11-07
主觀
心靈認知

我們人生、認知、價值觀，對於事物的感知，就是一張心靈地圖。

Start

探索地圖

打開地圖可會看到，成長家庭、家庭教育、學校教育，到價值觀養成，工作習慣的建立，家庭婚姻、養育教子，自我興趣發展，一系列有關於你的一切，你就能明白你擁有，與你缺失的。大概括，親友愛情、以及自我與職涯。

交流地圖

常有如下例子，與這個人就是尬聊，基本上雞同鴨講，或是與誰相談勝歡，把酒飲歌，或是有些牛人，什麼都能聊。人際關係交往中就是攤開地圖，互相審閱甚是比對的過程，你跟他都對金融有興趣，地圖上就是有金融部分重疊，重疊的部分越多，代表你們越相似，自然調得來。

完善地圖

20歲年輕小夥子，與60歲的老人，地圖裡應當有所不同，人救世會透過交流的過程中，相互模仿與學習，這塊缺口，我補一下，最終完善自我的價值觀。
地圖說白就是一種意識，也可以是本書、是張畫，只是就由一種形象讓我們更好理解，最後想說無論是什麼，要記得下筆的是你自己。
